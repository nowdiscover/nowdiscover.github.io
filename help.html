Authorize Magento for NowDiscover

This guide helps you to setup a dedicated user for NowDiscover in Magento so that both platforms can talk securely to each other.

To sign up your Magento shop you need to grant NowDiscover access to it. This guide lists what you need before you start. It then shows you how to create a dedicated user role and profile for NowDiscover in Magento so that both platforms can talk to each other in the most secure way.

The process will take you about 3-4 minutes.

Before You Start

Know the URL of your Magento Shop’s Admin Panel

To authorize NowDiscover you need to sign in to the Admin Panel of your Magento shop.

If your Magento shop is set up with standard settings its Admin Panel is accessible under the URL of your shop amended by the extension “/admin”. If for instance your shop can be found under “http://example-shop.com”, the Admin Panel is accessible under “http://example-shop.com/admin”.

As some Magento shops use a custom URL for the Admin Panel, the URL of your Admin Panel might differ from this scheme. If you have a dedicated person responsible to administrate your shop, she or he is probably the right person to talk to. In case you have lost this information you also can find the URL in the E-Mail that you have received as a confirmation right after Magento had been set up.

Know the URL of your Magento Shop’s Admin Panel

A Sign-In ID with administration rights allows you to create new user roles and profiles at the Magento Admin Panel.

The sign-in ID that had been used to install the Magento shop system is usually the sign-in ID with administration rights. If you have a dedicated person responsible to administrate your shop, she or he is probably the right person to talk to. In case you have lost this information you also can reset the password directly at the sign-in page of the Magento Admin Panel by clicking at “Forgot your password?”

Know your Magento Version

Currently Now Discover supports Magento shop system versions 1.8.x until 1.9.x and the corresponding sub versions.

Older versions or Magento 2 are not yet supported. You can securely check the version of your Magento shop system before you continue with the sign up process of NowDiscover. Simply sign in to your Magento Admin panel and check the information at the footer.

1. Navigate To User Roles at the Magento Admin Panel

  1. Go to the Admin Panel of your Magento shop
  2. Sign in
  3. Select at the top navigation dropdown: “System > Web Services > SOAP/XML-RPC - Roles”

2. Navigate To User Roles at the Magento Admin Panel

  1. Click at “Add new Role” (Upper Right Corner)
  2. Enter at “Role Name” “NowDiscover” and your password to authorize
  3. Change the focus of the table to “Role Resources” (Left Side Bar)
  4. Select at the dropdown “Resource Access” the entry “All”(Main Cell of Table)
  5. Click at the orange button “Save Role”(Upper Right Corner)

3. Create a Dedicated User for NowDiscover

  1. Select at the Top Navigation Dropdown: “System > Web Services > SOAP/XML-RPC - Users”
  2. Click at the orange button “Add new User”(Upper Right Corner)
  3. Enter at:

    User Name:nowdiscover

    First Name:NowDiscover

    Last Name:NowDiscover

    Email:support@nowdiscover.com

    Current Admin Password:Password of the Magento Sign-In ID with Admin rights

    API Key:Copy-Paste the API-Key generated by NowDiscover

    API Key Confirmation:Copy-Paste the same the API-Key to confirm

  4. Change the focus of the table to “User Role”(Left Side Bar)
  5. Select the radio button “NowDiscover”(Main Cell of Table)
  6. Click on the orange button “Save User”(Upper Right Corner)
  7. Click on “Log Out” to sign out from Magento(Upper Right Corner)
  8. Close the browser tab and return the tab with the NowDiscover Registration

Note:
If you accidently closed the tab with the NowDiscover sign up you can continue signing up by simply clicking here.
